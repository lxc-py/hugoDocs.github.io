训练教程

训练流程概述.仅提供训练宜美智AOI与AVI的AI训练
  • 1: 训练步骤
  • 2: 各类模型标签Json源码
  • 3: 模型验证

1 - 训练步骤

从创建项目到训练到验证的训练步骤.
  • 创建项目
  • 快速创建任务
  • 进行标注
  • 开始训练

创建项目

创建项目.具体请参阅创建项目

备注:因为需要该项目的数据参与训练,故创建项目时最好选择设备和工序/表面
各种工序所使用的标签的Json源码格式请参考各类模型标签Json源码

快速创建任务

  1. 设置检修软件AOI是AI工序选择 AVI是AI膜面选择,只有设置了才会导出图片
  2. 检修软件导出所需图片,导出路径为参数设置->缺陷报告路径下的Dataset文件夹的以AI工序选择的名称命名的文件夹内
    • F7判定所选图片为漏失并导出到路径中的aoi_missing文件夹中
    • F8判定所选图片为误判并导出到路径中的aoi_wrong文件夹中
  3. 在项目页面上动作菜单里点击快速创建任务按钮, 就会选择从检修导出的图片上传到服务器并创建任务,任务命名默认为当前日期

备注 从项目快速创建任务取决于项目的工序或膜面,比如,如果项目的工序是内层,则快速创建任务所选的图片是从{导出路径}/内层中读取所有图片,如果没有图片,则会提示错误

进行标注

标注请参考

  • 使用矩形标注
  • 使用多边形标注

标注完成之后在标注界面的菜单点击完成作业

开始训练

  1. 参数配置部分
    1. 选择设备 AOI/AVI
    2. 选择模型类型
    3. 完成前两步后会根据这两个属性筛选项目,然后选择项目后作为数据训练
    4. 显卡选择 分单卡训练和多卡训练
    5. 训练轮数 选择深度学习训练次数,训练轮数越多,通常学习越深,但到了一定轮数就会有极限
      通常200-300轮
  2. 上传模型配置
    1. 当前模型路径 指的是当前AI电脑所使用的模型路径,要以共享路径指出
    2. 主机名 AI电脑主机名
    3. 共享密码 AI电脑共享密码
    4. 模型名称 AI电脑所使用的模型名 通常选择模型属性后会自动生成模型名称
  3. 开始训练 上述参数配置好之后即可点击开始训练按钮
  4. 停止训练 如果中途想停止训练, 点击停止训练按钮,但训练结果将不会保存到服务器

3 - 模型验证

验证训练模型是否能识别出任务中图片的缺陷
  • 添加任务图片到验证集
  • 进行验证
  • 判定NG/OK
  • NG图片标注
  • 导出NG图片

添加任务图片到验证集

  1. 在导航栏点击验证进入验证界面,
  2. 选择模型属性后, 将会显示相对于的验证图片以及任务列表
  3. 点击加入验证集将该任务添加到验证图片集中

显示图片的格式是

如果AI未识别或者未参与验证, AI识别图将和原图一致

进行验证

跳转到训练界面选择设备和模型属性后点击开始验证 所有图片验证完成之后将会自动跳转到验证界面

判定NG/OK

当AI模型验证图片跳转到验证界面后,如果AI验证结果不符合人工预期, 可以在验证图片框里选择下拉框, 选择标记为NG

NG图片标注

当检查完验证图片之后,如果有NG图片,可以选择查看含有NG图片任务 然后跳转到相应的标注界面进行标注
