要在自训练平台上进行模型验证，您可能需要先导入训练模型并指定其参数。

要导入训练模型, 请在 **训练** 页面上单击 **+** 并
选择 **导入训练模型**.
![导入训练模型](/images/backup_model.png)

## 导入训练模型
1. 填入一下参数
![](/images/import_model_param.png)
| 参数          | 详述                                                                                                                                                                                                                                         |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 名称          | 提供**模型名称**                                            |
| 模型 | 选择训练模型工序             |
| 批量大小        | 训练时的批量大小(batch_size)                                                                                                                                                                                                  |
| 图像尺寸        |  数据集的图像尺寸.|
| 训练轮数        |  模型所在的epoch.|

2. 上传**模型文件**
3. 点击**确定**
